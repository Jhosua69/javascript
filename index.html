FACTORIAL

Explicación del cálculo del factorial:

El cálculo del factorial se realiza de la siguiente manera:

n = prompt("Digite n: "); solicita al usuario que ingrese un número 'n'.

const e = 2.71828182845; define la constante 'e' como la constante matemática 'e'.

base = 2; define la base para el cálculo del factorial.

nFac = Math.sqrt(2 * Math.PI * n) * Math.pow((n / e), base); calcula la aproximación del factorial utilizando la fórmula de Stirling.

Finalmente, console.log("Aproximación del factorial:", nFac.toFixed(2)); imprime la aproximación del factorial en la consola del navegador con dos decimales.
